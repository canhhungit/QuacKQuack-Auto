### Chức năng của extension

- Tự động bật game
- Tự động nhặt trứng
- Tự động mở trứng để full vịt
- Xoá vịt đểu để mở trứng xịn thay thế vịt đều
- Tự động mở vịt vàng
- Tự động reload lại trang nếu login hết hạn
